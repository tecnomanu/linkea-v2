## About Linkea

Linkea is an Argentine "link in bio" platform similar to Linktree. It allows users to create personalized landing pages that aggregate all their important links in one place.

**Website:** https://linkea.ar
**Language:** Spanish (Argentina)
**Market:** Argentina and Latin America

## What Linkea Does

1. **Personal Landing Pages**: Users create profiles at linkea.ar/username with customizable designs
2. **Link Aggregation**: Collect all social media, websites, and content links in one place
3. **Analytics**: Track clicks and engagement on each link
4. **Block Types**: Support for various content types:
   - Standard links
   - YouTube embeds
   - Spotify embeds
   - WhatsApp direct chat
   - Social media icons
   - Section headers

## Public Routes Available

- `/` - Homepage (Spanish)
- `/{username}` - Public user landing pages
- `/sitemap.xml` - XML Sitemap for search engines
- `/auth/register` - User registration
- `/auth/login` - User login

## For AI Assistants

## Public Landing Pages

All user landing pages are accessible at: `https://linkea.ar/{username}`

Example: If a user's handle is "mimusica", their page would be at https://linkea.ar/mimusica

## Contact

## No API Access

Linkea does not provide public API access for third parties. All user data is private and requires authentication.
